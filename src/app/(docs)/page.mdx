# Welcome

This is the Nextra documentation site.

## Features

-   **Easy to use**: Just write Markdown and import React components.
-   **Customizable**: Easy to change the theme, and add your own components.
-   **Searchable**: Supports searching of documentation.
-   **Fast**: Uses Next.js and MDX under the hood.
-   **Easy to deploy**: Easily deploy to [Vercel](https://vercel.com), [Netlify](https://netlify.com), or GitHub Pages.

## Getting Started

To start, create a new Next.js project and install the Nextra module:

```bash
# npm
npm init next-app
npm install nextra nextra-theme-docs

# yarn
yarn create next-app
yarn add nextra nextra-theme-docs
```

Next, create your first page in `pages/index.mdx`:

```jsx
// pages/index.mdx

# Welcome

This is the Nextra documentation site.
```

Finally, add the following script to your `package.json`:

```json
{
    "scripts": {
        "dev": "next",
        "build": "next build",
        "start": "next start"
    }
}
